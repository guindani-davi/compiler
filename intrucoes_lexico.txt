Encontrar Lexema
  Inicio
    lexema é inicializado com caractere de entrada
    Se caractere de entrada é letra Então: {
      Enquanto letra ou número Faça: 
        Concatenar no lexema
    }
    Senão 
      Se caractere de entrada é número Então: {
        Enquanto número ou 1o ponto Faça:
          Concatenar no lexema
      } 
      Senão 
        Se caractere de entrada é inicio de token Então: {
          Enquanto formar token Faça:
            Concatenar no lexema
        }
        Senão 
          o lexema é o caractere de entrada
    retornar lexema
  Fim
